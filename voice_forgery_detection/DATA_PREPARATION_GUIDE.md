# 📊 数据准备指南

## 什么是训练数据？

想象一下教小孩识别猫和狗：
- 我们需要给小孩看很多猫和狗的照片
- 告诉小孩"这是猫"、"这是狗"
- 小孩通过大量例子学会区分

我们的项目也一样：
- **真实语音**：真人说话的声音（相当于"猫"）
- **伪造语音**：AI生成的声音（相当于"狗"）
- **目标**：让计算机学会区分这两种声音

## 数据来源

### 真实语音数据
1. **LibriSpeech数据集**（推荐）
   - 包含大量真实人声
   - 免费下载
   - 质量高，标注准确

2. **VCTK数据集**
   - 多说话人语音数据
   - 适合训练多样性

### 伪造语音数据
1. **ASVspoof数据集**
   - 专门用于语音伪造检测
   - 包含多种伪造技术

2. **自生成数据**
   - 使用TTS工具生成
   - 如：VITS、Tacotron等

## 数据准备步骤

### 第一步：创建数据目录结构
```
data/
├── real/          # 真实语音
│   ├── speaker1/
│   ├── speaker2/
│   └── ...
└── fake/          # 伪造语音
    ├── tts1/
    ├── tts2/
    └── ...
```

### 第二步：数据格式要求
- **音频格式**：WAV（推荐）、MP3
- **采样率**：16kHz（标准）
- **时长**：3-10秒（每段音频）
- **数量**：每类至少100个样本

### 第三步：数据预处理
1. **格式统一**：所有音频转换为WAV格式
2. **采样率统一**：统一为16kHz
3. **长度标准化**：裁剪或填充到相同长度
4. **质量检查**：去除损坏的音频文件

## 初学者快速开始

如果您是初学者，我建议：

1. **先使用小数据集**（每类50个样本）进行测试
2. **使用在线TTS工具**生成伪造语音
3. **录制自己的声音**作为真实语音样本

## 数据质量检查

### 音频质量指标
- **信噪比**：>20dB（声音清晰）
- **无静音段**：避免长时间静音
- **无失真**：避免爆音、杂音

### 数据平衡
- 真实语音和伪造语音数量要大致相等
- 每个说话人的样本数量要平衡
- 避免数据倾斜
