# 🎓 初学者完整使用指南

## 📚 项目概述

这个项目是一个**语音伪造检测系统**，能够识别音频是否为AI生成的伪造语音。就像识别假币一样，我们的系统能识别"假声音"。

### 🎯 项目目标
- **输入**：音频文件（WAV、MP3等）
- **处理**：AI分析音频特征
- **输出**：判断结果（真实/伪造）+ 置信度

## 🚀 快速开始（3步完成）

### 第一步：安装环境

1. **安装Python**
   - 访问 [Python官网](https://www.python.org/downloads/)
   - 下载Python 3.8或更高版本
   - 安装时**务必勾选**"Add Python to PATH"

2. **安装依赖库**
   ```bash
   # 在项目目录下运行
   pip install -r requirements.txt
   ```

3. **验证安装**
   ```bash
   python -c "import librosa, sklearn, flask; print('✅ 安装成功')"
   ```

### 第二步：准备数据

1. **创建数据目录**
   ```
   data/
   ├── real/     # 真实语音（放这里）
   └── fake/     # 伪造语音（放这里）
   ```

2. **准备音频文件**
   - **真实语音**：录制真人说话，或下载真实语音数据集
   - **伪造语音**：使用TTS工具生成，或下载伪造语音数据集
   - **数量要求**：每类至少50个文件（初学者）
   - **格式要求**：WAV、MP3、M4A、FLAC

3. **数据质量检查**
   - 音频清晰，无杂音
   - 时长3-10秒
   - 文件大小<16MB

### 第三步：运行项目

**方法一：一键运行（推荐）**
```bash
python run_project.py
```

**方法二：分步运行**
```bash
# 1. 数据预处理
python data_preparation.py

# 2. 特征提取
python feature_extraction.py

# 3. 模型训练
python train_model.py

# 4. 启动Web应用
python webapp/app.py
```

## 🎮 使用方法

### Web界面使用

1. **启动应用**
   ```bash
   python webapp/app.py
   ```

2. **访问网页**
   - 打开浏览器访问：http://localhost:5000
   - 上传音频文件
   - 查看检测结果

### 命令行使用

```bash
# 检测单个音频文件
python predict.py your_audio.wav
```

## 🔧 技术原理（简化版）

### 1. 特征提取
- **MFCC特征**：模拟人耳听觉特性
- **频谱特征**：分析声音的频率分布
- **节奏特征**：分析声音的节拍和节奏

### 2. 机器学习
- **训练过程**：让计算机学习真实和伪造声音的区别
- **预测过程**：输入新声音，输出判断结果

### 3. Web应用
- **前端**：用户上传界面
- **后端**：AI模型推理
- **可视化**：显示频谱图和结果

## 🎯 常见问题解决

### 问题1：安装失败
**解决方案**：
```bash
# 升级pip
python -m pip install --upgrade pip

# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/
```

### 问题2：数据不足
**解决方案**：
- 使用在线TTS工具生成伪造语音
- 录制自己的声音作为真实语音
- 下载公开的语音数据集

### 问题3：模型训练失败
**解决方案**：
- 检查数据质量
- 确保数据平衡（真实和伪造数量相近）
- 检查Python版本和依赖库

### 问题4：Web应用无法访问
**解决方案**：
- 检查端口5000是否被占用
- 尝试访问 http://127.0.0.1:5000
- 检查防火墙设置
