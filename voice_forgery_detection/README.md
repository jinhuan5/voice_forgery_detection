# 语音伪造检测系统 (Voice Forgery Detection System)

## 📋 项目概述

本项目是一个智能的语音伪造检测系统，能够准确识别由TTS（文本转语音）、Voice Clone、Deepfake等技术生成的伪造语音。系统基于高质量的语音特征和机器学习算法，为用户提供可靠的语音真伪判断。

### 🎯 系统目标

- **输入**：音频文件（WAV、MP3等格式）
- **处理**：提取语音特征 → 模型分析 → 真伪判断
- **输出**：检测结果（真实/伪造）+ 置信度评分

## 🏗️ 系统架构

### 核心工作流程

```
音频输入 → 特征提取 → 模型预测 → 结果输出
    ↓         ↓         ↓         ↓
  音频文件   MFCC/频谱图  分类模型   真伪+置信度
```

### 技术实现方案

#### 1. 数据准备
- **真实语音**：来自公开数据集的高质量人声样本
- **伪造语音**：来自ASVspoof数据集和TTS工具生成的样本
- **数据质量**：经过人工筛选，确保数据质量

#### 2. 特征提取
- **主要特征**：MFCC（梅尔倒谱系数）、频谱特征、节奏特征
- **提取工具**：librosa音频处理库
- **采样率**：16kHz（标准语音处理采样率）
- **特征维度**：52维特征向量

#### 3. 模型训练
- **算法**：随机森林、梯度提升、SVM、逻辑回归
- **模型选择**：自动选择最佳性能模型
- **输出类别**：真实(0) / 伪造(1)

#### 4. 模型评估
- **评估指标**：准确率(Accuracy)、AUC、混淆矩阵
- **交叉验证**：确保模型泛化能力
- **性能监控**：实时模型性能评估

#### 5. Web应用
- **前端**：HTML + JavaScript（音频上传界面）
- **后端**：Flask（模型推理服务）
- **可视化**：频谱图展示、检测结果可视化

## 🚀 快速开始

### 环境要求
- Python 3.7+
- 依赖库：librosa, scikit-learn, flask, numpy, matplotlib

### 安装依赖
```bash
pip install -r requirements.txt
```

### 使用步骤

#### 方法1：一键启动（推荐）
```bash
python run_project.py
```

#### 方法2：分步执行
1. **数据预处理**
```bash
python data_preparation.py
```

2. **训练模型**
```bash
python train_model.py
```

3. **启动Web应用**
```bash
cd webapp
python app.py
```

4. **访问应用**
打开浏览器访问 `http://localhost:5000`
